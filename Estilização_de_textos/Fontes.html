Trabalhando com fontes

Font-family - Em geral essa propriedade muda o formato da fonte

Font-weight - Essa propriedade muda o peso da fonte, deixando ela mais grossa ou fina

Font-style - Essa propriedade muda o estilo da fonte

Font-size - Essa propriedade muda o tamanho da fonte

Font-web - Em caso do cliente não ter a fonte usada na página, ela ainda irá aparecer para ele, pois ela estará sendo tirada da web

Mais estilos para os textos

Font-variant - Faz variações na apresentação da fonte

Font-stretch - Alargamento ou encolhimento da fonte

Letter-spacing - Define o espaçamento entre os caracteres

Word-spacing - Define o espaçamento entre palavras

Line-height Define os espaços entre linhas

Text-transform - Transforma o texto

Text-decoration - Aparência decorativa de um texto

Text-align - Alinhamento de um texto

Text-shadow - Sombra aplicada a um texto

Font - É usado para fazer o shorthand
